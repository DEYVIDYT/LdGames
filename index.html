Guia de Uso da Cloud LdGames

⚠️ Aviso Importante:

A cloud do LdGames deve ser utilizada exclusivamente para uso pessoal e somente para armazenar cópias de jogos que você já possui legalmente.

Estrutura do JSON da Cloud

Abaixo está um exemplo do formato correto para estruturar seus dados na cloud:

{
  "name": "MinhaCloud",
  "downloads": [
    {
      "title": "Super Mario World",
      "fileSize": "2.3MB",
      "uris": [
        "https://meuservidor.com/downloads/mario-world",
        "https://backup.com/downloads/mario-world"
      ],
      "uploadDate": "2023-12-25"
    },
    {
      "title": "Sonic the Hedgehog",
      "fileSize": "1.8MB",
      "uris": [
        "https://meuservidor.com/downloads/sonic",
        "https://backup.com/downloads/sonic"
      ],
      "uploadDate": "2023-12-26"
    }
  ]
}

Explicação dos Campos

  • name: Nome identificador da sua cloud pessoal
  • title: Nome do jogo armazenado
  • fileSize: Tamanho do arquivo do jogo
  • uris: Links para download do jogo (recomenda-se sempre ter mais de um para backup)
  • uploadDate: Data em que o arquivo foi enviado para a cloud

Lembre-se:

  • Mantenha suas ROMs organizadas e bem descritas
  • Sempre inclua links alternativos para backup
  • Use apenas para armazenar jogos que você possui legitimamente
  • Não compartilhe sua cloud com terceiros
